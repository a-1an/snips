## windows essentials 
- <kbd>![Windows Key][newwinlogo]</kbd> + <kbd>v</kbd> opens clipboard history. Use it to copy and paste multiple items.
- <kbd>![Windows Key][newwinlogo]</kbd> + <kbd>G</kbd> opens game bar. use it to capture screenshots.
